# Overview

I created this project to practice setting up a C# development environment and learn how to use Git and GitHub to manage my code. This is the first step toward building a software portfolio and becoming more comfortable with the software development process.

This software is a simple C# console application that displays "Hello, World!" along with a short introduction.

The purpose of this project is to gain experience creating, running, and publishing a basic program while learning the tools that will be used throughout the course.
